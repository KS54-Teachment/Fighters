#  Генератор паролей

Простое, красивое и безопасное веб-приложение для создания надёжных паролей. Работает прямо в браузере, не хранит и не передаёт ваши пароли.

## Особенности

-  **Настройка длины** пароля от 4 до 32 символов
-  **Выбор типов символов**: заглавные, строчные, цифры, спецсимволы
-  **Индикатор надёжности** пароля в реальном времени
-  **Копирование в буфер** одной кнопкой
-  **Адаптивный дизайн** — работает на всех устройствах
-  Мгновенная генерация без перезагрузки страницы
-  Вся логика выполняется локально — конфиденциальность гарантирована

##  Демо

Откройте `index.html` в любом современном браузере или [разместите на GitHub Pages](https://docs.github.com/ru/pages).

##  Использование

1. Выберите длину пароля с помощью ползунка
2. Отметьте нужные типы символов
3. Нажмите «Сгенерировать» или настройте параметры — пароль обновится автоматически
4. Скопируйте пароль кнопкой с иконкой 

##  Рекомендации

- Используйте пароли длиной **от 12 символов**
- Включайте **все типы символов** для максимальной защиты
- Не используйте один пароль для разных сервисов
- Храните пароли в менеджере паролей (Bitwarden, KeePass, 1Password)
